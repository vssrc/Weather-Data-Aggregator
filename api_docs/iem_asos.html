  2. Download ASOS 1 minute interval data

Each month, the National Climatic Data Center provides an archive of one minute interval data from the ASOS sites in the United States. Unfortunately, the data format of this information is brutal to work with. Fortunately for you, the IEM processes this information and provides it for download here. The archive is available up until: 12 Nov 2025. There is a different interface to download the routine hourly observations from this network and for sites over the entire globe.

Backend Documentation exists for those that wish to script against this service.

1. Select Station:


Select One or More or All stations in the network.

2. Timezone of Observations:

The timestamps used in the downloaded files will be set in the timezone you specify.

3. Select Start/End Time:


This timestamp is in the timezone you selected above.

5. Data Sampling?


Data is potentially available every minute, but you don't have to download it at that frequency. This setting does not bin the values, but rather filters based on your interval of choice.
